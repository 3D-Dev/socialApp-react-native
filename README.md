# Project Title

Mocha App

## Getting Started

Please follow the below instruction get it running.

### Prerequisites

```
node
```

### Installing

```
npm install -g yarn
npm install -g expo-cli
yarn
```

### Running

```
yarn start
```

### Testing
```
yarn lint // Run eslint (all js/jsx files)
yarn lint-fix // Fix linting issues
```

### Translate
```
yarn translate
```
